Cómo obtener la clave de Google AI

Es gratis y lleva solo 3 minutos. Seguí estos pasos:

  1. Abrí el navegador y entrá a aistudio.google.com
  2. Iniciá sesión con tu cuenta de Google (la misma del Gmail)
  3. Hacé clic en el botón "Get API Key" que está arriba a la izquierda
  4. Hacé clic en "Create API key"
  5. Copiá la clave que aparece (empieza con "AIza...")
  6. Volvé aquí y pegala cuando el sitio te la pida

✓ Es completamente gratuita · ✓ No se comparte con nadie · ✓ Se guarda solo en este dispositivo
